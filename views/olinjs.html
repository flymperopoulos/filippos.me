Web Development

OlinJS is a student-taught course at Olin on web development using Node.js. The class had several labs were we were asked to create simple web apps that utilized new features of the stack available to us. We started off using Node.js with jQuery and Handlebars and as the semester progressed we utilized JS frontend frameworks, such as AngularJS, and publicly accessible APIs on the web to make cool applications. We used MongoDB as our database for all the above.

My final project for OlinJS was Boolio a startup that a team of Olin students and I are currently pursuing.

In the Spring of 2016, I will be teaching this class to the next generation of OlinJS students!
