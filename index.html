Shooter Platform

Python OpenCV Game using the Pygame library

For Olin's Software Design class two friends of mine and I designed a really cool OpenCV-based Python game. Approximation of the 3D behavior of the game is achieved via the enemies descending motion along the field. The game incorporates the enemies' ability to shoot at you, while you target them with a custom made LED gun. Tracking the color of the LED gun and eliminating the effects of the surrounding environment was challenging but yielded a remarkable result. Awesome gun updates are available too!

The code of project is open source and can be found in the following github repo!

  • Date: Spring 2014
  • Software: Python, Pygame, OpenCV

Heat Signature Mapping

Real World Measurements Final Project

For my final project for the Real World Measurements class I designed a heat signature mapping device. For this to be done I used the Arduino microcontroller and the MLX90614 sensor. This sensor has a low noise amplifier built into it, along with a 17-bit ADC, which allows very accurate and high resolution temperature readings.

Several results were produced that can be found in the repository of the project on github here!

  • Date: Spring 2014
  • Hardware: Hobby Servo, IR MLX90614 Sensor
  • Software: Arduino C, MATLAB, Python

Twitter Sentiment Analysis

Used Python to Analyze the Oscars 2014 Aftermath

Following the Oscars, we were really interested in mining and further analyzing data from Twitter feeds to determine when and what the general public was discussing during the event. The general approach was to get access to such data, and deduce how and to what extent people talk during, and at the aftermath of the event. Instead of generally looking into the Oscars, we zoomed into the trends of individuals that were nominated for Best Actor, Best Movie etc.

For more infromation visit the github repository for the project here!

  • Software: Python (pattern.web, pattern.en)

Basketball Analysis and Simulation

Integration of MATLAB and Tracker to analyze and predit basketball behavior.

This was the final project for the "Mechanics: Modelling and Simulation Approach" class. With my teammate James Jang we studied the sport of basketball and investigated the nature of swishshots and banked shots. We developed a 3D model of the sport and conducted motion tracking analysis to validate our model. We further considered rim and backboard interactions and developed some very important figures of merit, including frontspin, backspin, and no spin shots.

All relevant documents and code are available on our github repository!

  • Date: Spring 2014
  • Software: MATLAB, Tracker
